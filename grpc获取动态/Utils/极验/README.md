# 极验V3点选验证码

>### 实现思路
> 1.采用DDDDOCR识别区块 \
> 2.通过clip大模型或其他方法获取图像embedding，使用cosine相似度判断是否为同一图像，生成成本矩阵\
> 3.使用匈牙利算法，对成本矩阵进行匹配，从而实现点选验证码的识别。
